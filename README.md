# Projet API Blog

Ce projet consiste à développer une API backend permettant de gérer des articles de blog.

## Technologies utilisées
- Java
- SQLite
- Postman

## Fonctionnalités
- Créer un article
- Lire les articles
- Modifier un article
- Supprimer un article
